# HMMER_API
 Request and retireve data from the HMMER website


https://hmmer-web-docs.readthedocs.io/en/latest/api.html

HMMER je service k ti omogoča iskanje proteinskih domen na proteinu, nazaj ti vrne podatke o poravnavi med domeno in proteinom.
Blo bi kul če se uporablja hmmscan ampak ne vem morem pogledat kak je endpoint.. Malo me jebe ker so examples v API dokumentaciji v JS.
